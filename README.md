# Image Classification Workflow (Group Coursework)

## Project Overview
This repository contains a group coursework project for image classification
on the CIFAR-10 dataset.

The project focuses on comparing different machine learning and deep learning
approaches (MLP, CNN, and SVM) within a complete and consistent machine learning
workflow, including preprocessing, model design, and hyperparameter tuning,
evaluation, and reporting.

## Dataset
- CIFAR-10 image dataset
- 60,000 colour images (32Ã—32 pixels) across 10 balanced classes
- Standard train/validation/test split

## Methods
The following models were implemented and compared:
- Multilayer Perceptron (MLP)
- Convolutional Neural Network (CNN)
- PCA-based RBF Support Vector Machine (SVM)

Models were evaluated using validation-based hyperparameter tuning and
final testing on a held-out test set. Performance was analysed using accuracy,
Macro-F1, confusion matrices, and training runtime.

## Key Takeaways
- A clear and well-organised machine learning workflow is as important as model performance
- Explaining *why* modelling choices are made is critical for meaningful evaluation
- Consistency in preprocessing, evaluation, and presentation is essential when integrating work from multiple contributors
